Sobre mí

Estudiante apasionado de Desarrollo Frontend con un firme compromiso por crear experiencias web atractivas y funcionales. Poseo sólidos conocimientos en tecnologías clave para el frontend, así como una habilidad demostrada para colaborar eficientemente en equipos de desarrollo. Con una mentalidad orientada a la resolución de problemas y un enfoque proactivo, estoy constantemente buscando oportunidades para mejorar y expandir mis habilidades en el campo.

MusicAudio

Web ficticia creada para incorporar los primeros conocimientos que adquirí en HTML y CSS. El proyecto emula una tienda donde se exhiben distintos instrmentos musicales, en este caso específico, guitarras acústicas.

Ver proyecto
Iguana Page

Web ficticia creada para incorporar adquiridos en HTML, CSS y Flexbox. El proyecto representa un tipo de wiki donde se exponen las principales características de este animal y fotografías. Además se utiliza por primera vez íconos de Font Awesome.

Ver proyecto
Cuppon

Web ficticia creada para incorporar adquiridos en HTML, CSS, Flexbox y Bootstrap. El proyecto presenta una página web donde se venden cupones para distintos servicios. El proyecto fue realizado de manera responsiva, para visualizarlo en formato web de escritorio y mobile.
